명성 홈페이지 리뉴얼 회의록

회의 주제 : 명성 홈페이지 리뉴얼 방향 및 구조 논의

목적 : 레거시 공사업 이미지를 줄이고, AI·비전·로봇 중심의 기술기업 이미지로 재정의

참석자 : 발화자 1(명성 측), 발화자 2(홈페이지 제작사), 발화자 3(명성 실무/기획)

형태 : 대면 미팅 (홈페이지 레퍼런스 사이트 검토 및 PPT 기반 논의)

1. 리뉴얼 전체 방향

  • 기존 레거시 사업(전기공사, 자동제어 공사, 유통 등)은 홈페이지 메인에서 최대한 비노출.
  • 신규 이미지는 AI, 비전, 로봇, 예지보전을 핵심 키워드로 잡고 “AI 기술 회사” 느낌을 강하게 전달.
  • 슬로건 방향 : “From Control to Intelligence” 컨셉 유지, 문구는 제작사가 다듬어서 적용.
  • 홈페이지 제작 목적:
    • ① 회사 대내·외 이미지 전환 (직원/고객에게 사업 전환 방향 공유 – 연말 행사·송년회 등에서 활용)
    • ② 내년 IR 및 투자유치 시 “서비스업·공사업체”가 아닌 “AI 기술 기업”으로 보이도록 브랜딩.

2. 정보 구조 및 메뉴 구성

  • 전반적인 구조는 페스토(Festo) 레퍼런스 페이지처럼 단일 롱 스크롤 원페이지 구성.
    • 상단 메뉴 클릭 시 해당 섹션으로 스크롤(드롭다운 스크롤) 이동.
    • 여러 서브페이지로 쪼개지 않고, 한 페이지 안에서 전체가 보이도록 구성.
  • 상단 메뉴(예시):
    • 회사소개
    • 제품소개 (기존 “솔루션” 명칭 → “제품소개”로 변경)
    • 인재채용
    • 문의하기(별도 페이지 대신, 하단 섹션 및 챗봇 활용)
  • 필수 섹션:
    • 메인 비주얼 & 슬로건
    • 핵심 사업 영역 : AI / 예지보전 / 비전 / 로봇 / (최소한의 레거시)
    • 제품 소개 : 스마트 진동 센서(쏘지니), 알람 발생기, 예지보전 솔루션 등
    • 회사 연혁(레거시 중 필요한 부분만 선별)
    • 인증·특허 현황
    • 협력사 로고 영역
    • 인재채용(요약 + 채용 바로가기 버튼)
    • 문의/챗봇, 주소 및 오시는 길

3. 디자인 및 색상 방향

  • 전반적 키워드 : 심플, 밝음, 군더더기 없는 레이아웃.
  • 배경 :
    • 기본 배경은 화이트(하얀색) 기반의 밝은 톤.
    • 전시회용 보라색, 너무 어두운 톤 등은 배제.
  • 포인트 컬러 :
    • 기본은 푸른 계열(블루/네이비·스카이블루) 중심으로 재정의.
    • 현재 초록 계열 요소는 점차 줄이고, 명성이 가진 “차분한 푸른 톤”으로 통일.
  • 컴포넌트 스타일:
    • 동그라미, 과도한 아이콘 장식 최소화.
    • 텍스트 위주 + 간단한 아이콘/이미지로만 강조.
    • 섹션 타이틀과 짧은 설명, 관련 이미지 1~2장 정도로 깔끔하게 구성.
  • 메인 비주얼:
    • 회사 전경/공장 동영상(2분반짜리) 별도 제작 중 – 12월 말 완성 예정.
    • 완성 전까지는 밝은 톤의 임시 배경 이미지/영상을 활용 (토스, 페스토 등처럼).

4. 섹션별 구체 논의 내용

4-1. 메인 섹션

  • 구성 요소:
    • 심플한 메인 카피(슬로건) + 짧은 서브카피.
    • AI·비전·로봇을 연상시키는 이미지/영상 배치.
    • 불필요한 네모 박스, 컬러 블록 최소화.
  • 참고 사이트:
    • 페스토(Festo) 메인(어바웃) 페이지 스크롤 흐름 레퍼런스.
    • 토스 스타일의 깨끗하고 밝은 인터페이스.

4-2. AI / 예지보전 섹션

  • 명성이 AI를 활용한 예지보전 솔루션을 제공한다는 점을 가장 먼저 부각.
  • 콘텐츠 아이디어:
    • “AI를 활용한 예지보전 솔루션” 설명 텍스트.
    • AI PEMS(전력 모니터링 시스템) 등 관련 이미지/도식.
    • 에너지 효율, 고장 예측, 생산성 향상 등 핵심 가치 제시.
    • 챗봇/AI 시스템 적용 사례(원 대표와 협의 후 사용 가능한 내용만 노출).
  • 대시보드 화면, 모니터링 UI 등은 추후 명성 측에서 영상/이미지 전달 예정.

4-3. 스마트 진동 센서 & 예지보전 솔루션(제품 섹션)

  • 주요 제품:
    • 스마트 진동 센서(쏘지니 스마트 센서)
    • 알람 발생기
    • 예지보전 통합 모니터링 솔루션(진이뷰 등)
  • 구성:
    • 제품 4장 내외 사진을 단순 그리드(2×2 등)로 배치.
    • 각 사진 아래 간단한 제품명/한줄 설명.
    • 기존 홈페이지의 관련 텍스트는 필요한 범위에서 가공하여 재사용.
    • 진이뷰 동영상 제공 예정 → 썸네일/짧은 루프 영상으로 일부 활용.

4-4. 비전/로봇 섹션

  • 비전:
    • 비전 카메라/검사 장비 사진 1~2장 사용 (JST 홈페이지 등 레퍼런스 참고).
    • “비전 검사 시스템”, “불량 검출·품질 향상” 등 키워드 중심으로 간단 설명.
  • 로봇:
    • 이노벤스(산업용 로봇) 관련 이미지 활용.
    • “설계 → 시공 → 제작 → 시운전” 흐름을 간단한 다이어그램이나 텍스트로 표현.

4-5. 회사소개 / 연혁 / 인증 / 협력사

  • 회사소개:
    • 기존 인사말, 장문 텍스트는 과감히 축소.
    • “우리는 AI/비전/로봇 기반의 스마트 제조 솔루션 기업”이라는 현재 정체성을 중심으로 재구성.
  • 연혁:
    • 레거시(전기공사 위주) 연혁은 최대한 정리·축소.
    • AI/비전/로봇/예지보전과 관련된 연혁을 앞으로 강조.
    • 기존 PPT에 정리된 연혁 자료를 가로 레이아웃으로 단순화하여 적용.
  • 인증/특허:
    • 현재 제작사의 홈페이지 구조(인증서/특허 탭 클릭 시 이미지 노출)를 유지.
    • 다만, 페이지 안에서 곧바로 이미지가 보이도록 단순화(버튼 클릭 구조는 유지).
    • 이미지는 업스케일링/리터칭하여 가독성 향상.
  • 협력사:
    • 협력사 로고를 한 줄 또는 슬라이더/애니메이션 형태로 표시.
    • 로고 수에 따라 1~2줄 구성 선택.

4-6. 인재채용 / 문의

  • 인재채용:
    • 기존처럼 사람인/채용 사이트 링크를 걸더라도, 별도 페이지 대신 한 블록(한 칸)만 구성.
    • “인재상 + 채용 바로가기 버튼” 정도만 표시.
    • 상세 공고 리스트, 공지 등은 노출하지 않음.
  • 문의:
    • 별도의 긴 ‘문의하기’ 폼 페이지는 생략.
    • 하단에 연락처·주소·오시는 길만 간단 기재.
    • 실제 문의는 챗봇을 통해 유도 (챗봇이 문의 방법 안내).

5. 제외/축소하기로 한 요소

  • 유통 관련 사업 영역 – 홈페이지에서는 완전히 제외.
  • 레거시 공사 실적(전기공사, 자동제어 공사 등) – 별도 상세 리스트는 넣지 않고, 필요 시 1장 수준으로만 요약.
  • 채용공고·공지사항 – 홈페이지에서 공고 목록 운용은 하지 않음.
  • 지나치게 어두운 배경, 복잡한 그래픽, 과한 애니메이션 – 최소화.

6. 자료 제공 및 역할 분담

항목 내용 담당 비고
메인 슬로건 확정 “From Control to Intelligence” 기반 문구 정리 및 한글/영문 최종 확정 명성 + 제작사 기존 PPT의 빨간색 문구 활용
매출·실적 자료 최근 연도 기준 매출 실적 정리 (연도별 그래프용) 명성 3년치 정도, 레거시 포함하되 표현은 단순화
연혁 정리 레거시 사업 중 필요한 연혁만 선별 + AI/비전/로봇 관련 연혁 추가 명성 PPT 형태 자료 제공 예정
AI·예지보전 이미지/영상 스마트 진동 센서, 예지보전 대시보드, 진이뷰 영상 일부 명성 동영상 제공 후 제작사에서 캡처/편집
비전/로봇 이미지 비전 검사 장비, 로봇 시스템(이노벤스 등) 사진 명성 또는 레퍼런스 사이트 기반으로 제작사에서 구성
인증·특허 스캔본 인증서·특허 이미지 원본 명성 제작사는 업스케일링 및 정리
뉴스·보도자료 AI/비전/예지보전 관련 기사/보도자료 명성 브로슈어/홈페이지 양쪽 활용 가능

7. 일정 및 다음 미팅

  • 마감 이미지 다음달 22일 행사에서 홈페이지를 활용할 수 있어야 함.
    • 그 전까지 2회 정도 미팅 필요(중간점검 + 최종 조정).
  • 다음 미팅(1차 중간 점검)
    • 시기 : 12월 첫째 주 월요일 11시 (음성 미팅 예정).
    • 형태 : 대면 또는 화상회의 (헝가리 출장이 잡힐 경우 화상으로 전환).
    • 내용 :
      • 원페이지 레이아웃 1차 시안 확인.
      • 메인 섹션/AI·예지보전/비전/로봇/제품소개 구성 검토.
      • 색상/타이포/이미지 톤 조정.
  • 동영상(회사 소개 메인 영상)은 12월 말 완성 예정이며, 현재는 임시 밝은 이미지/간단 루프 영상으로 대체 후 추후 교체.

8. 종합 정리

  • 홈페이지는 “명성 = AI/비전/로봇·예지보전 회사”라는 메시지를 가장 앞에 세우는 방향으로 리뉴얼.
  • 구조는 레퍼런스 사이트(페스토, 토스 등)처럼 심플한 원페이지 + 섹션 스크롤 이동을 기본으로 함.
  • 레거시 실적/유통/채용공고 등은 과감히 빼고, 핵심 기술·제품·인증·협력사를 짧고 강하게 보여주는 구성을 지향.
  • IR·투자, 브로슈어, 연말 행사 등 다양한 용도로 재활용할 수 있도록 텍스트와 시각자료를 함께 정리.
